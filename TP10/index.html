TPS Calculatrice







Ce TP m’a permis de concevoir une petite calculatrice en ligne en combinant HTML, CSS et JavaScript. J’y ai découvert comment interagir avec le DOM, gérer des événements utilisateurs, réaliser des calculs de base, vérifier si un nombre est pair ou impair, et échanger des valeurs. J’ai également appris à intégrer un script JavaScript à une page HTML et à ajouter une fonction permettant de réinitialiser le formulaire. Cette expérience m’a aidé à mieux saisir le fonctionnement et la communication entre le code et l’interface d’un site web.
